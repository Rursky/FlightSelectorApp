# FlightSelectorApp

LIVE DEMO: https://rursky.github.io/FlightSelectApp/

1. Clone

$ git clone https://github.com/Rursky/FlightSelectorApp

2. Run

To run install locally

$ npm install
$ npm start

3. DevServer

to run devserver on http://localhost:9000/

$ npm run start:dev

4. Production build

$ npm run build
